# Play12 Frontend

Este diretório contém o frontend React do projeto Play12. Todos os arquivos do antigo diretório `frontend` foram migrados para cá, mantendo a mesma estrutura de subpastas (`public`, `src`, etc).

## Estrutura
- `public/` — arquivos estáticos e imagens
- `src/` — código-fonte React
- `package.json`, `tsconfig.json` — configuração do projeto

## Como rodar
```bash
cd src/frontend
npm install
npm start
```
